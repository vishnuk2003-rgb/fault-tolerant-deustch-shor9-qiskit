# Fault-Tolerant Deutsch Algorithm using Shor's 9-Qubit Code

This repository contains the implementation of a fault-tolerant version of the Deutsch algorithm for the oracle f(x)=0 using Shor's 9-qubit quantum error-correcting code.

## Features
- Logical qubit encoding using Shor's [[9,1,3]] code
- Fault-tolerant circuit construction
- Deutsch oracle for f(x)=0
- Error detection/correction simulation
- Qiskit implementation
